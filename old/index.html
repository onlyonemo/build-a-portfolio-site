Project 1: A Portfolio Page

This was my first project in this class. I learned a lot about HTML and CSS. Some of the things I learned include:

  • Googlefonts need to be applied using both the CSS and the HTML on the appropriate Googlefont page (eg. not just HTML that goes in the "head").
  • Text weights can be dictated quickly by the h1, h2 etc styles, and can be adjusted later with CSS.
  • When I'm confused about why my layout isn't working...go to the W3C HTML validator! That saved me from going insane.
  • Adding a Bootstrap modal seemed cool, but without the trigger being a button (and being an image instead) you can't tell it is clickable. Bad UX.
